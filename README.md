
# Summer Reading 2026 Button Exchange Explorer

This is a little app to showcase all of the cool buttons and participants for this year's Summer Reading Button Exchange from the [Guild of Library Makers](https://guildoflibrarymakers.com/). 

Pushes to `main` are currently deploying to: **https://summer-reading-buttons.netlify.app/**

## About the App

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

### Run it Locally

If it's your first time here, clone the project and install packages with `npm install .` (or `pnpm install .`, etc.).

Run the development server with:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result!
